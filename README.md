# ETF Price Monitor for BMO Coding Assessment

A full-stack web application that reconstructs the historical performance of an ETF based on constituent weights. It features a Python/Flask backend for data processing and a React frontend for interactive visualization.

## Tech Stack
- **Frontend:** React, Recharts, Axios
- **Backend:** Python, Flask, Pandas

## Prerequisites
- Node.js
- Python

---

## Setup & Running

### 2. Start the Backend
The backend runs on Port **5001**.

Open a terminal and run:

```bash
cd backend

# Install dependencies
pip install -r requirements.txt

# Run server
python app.py

```
### 1. Start the Backend
The backend runs on Port **5001**.

cd etf-dashboard
